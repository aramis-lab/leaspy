# Repository Traffic Dashboard

This dashboard displays weekly traffic statistics for the Leaspy repository, automatically updated every Monday by GitHub Actions.

## Views and Visitors

Track how many times the repository has been viewed and by how many unique visitors.

![Repository Views Timeline](./_static/traffic/views_timeline.png)

## Clones Activity

Monitor repository clones, which typically indicate active usage by developers.

![Repository Clones Timeline](./_static/traffic/clones_timeline.png)

## Weekly Summary

Aggregated weekly statistics comparing views and clones over time.

![Weekly Traffic Summary](./_static/traffic/weekly_summary.png)

## Traffic Sources

Discover where your visitors are coming from.

![Top Referrers](./_static/traffic/top_referrers.png)

## Popular Content

See which pages and files are most frequently accessed.

![Popular Paths](./_static/traffic/popular_paths.png)

---

**Last Updated:** Automatically updated every Monday at 00:00 UTC

**Data Source:** GitHub Traffic API (past 14 days rolling window)

**Note:** GitHub only retains traffic data for the past 14 days. This dashboard preserves historical data by collecting it weekly.

For more information about the traffic monitoring system, see the [Traffic Monitoring Documentation](docdev/traffic_monitoring.md).
